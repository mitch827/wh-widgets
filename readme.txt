== Description ==

This plugin add a collection of widgets like a Foundation compatible side-nav, footer widgtes and others. Superpowered by **[Web Heroes.](http://www.webheores.it)**

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wh-widgets.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Setup options in menu.
